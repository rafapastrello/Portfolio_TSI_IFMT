Sobre a Disciplina

Fundamentos de Front End é a disciplina onde teoria e criatividade se encontram. Aprendemos a estruturar páginas web com HTML5 semântico, a estilizá-las com CSS3 moderno (incluindo Flexbox, Grid e animações), e a adicionar interatividade com JavaScript puro.

O foco é construir uma base sólida de desenvolvimento web, entendendo como o navegador interpreta e renderiza código, boas práticas de acessibilidade, design responsivo e experiência do usuário.

Ementa resumida

✓ Estrutura do documento HTML
✓ Seletores e especificidade CSS
✓ Box Model e layout
✓ Flexbox e CSS Grid
✓ Responsividade e Media Queries
✓ DOM e eventos JS
✓ Formulários e validação
◎ Introdução a frameworks
